Docker

Ubuntu Pre-requisites if on Ubuntu 20.04. Docker Desktop not supported in 18.04
1. sudo apt install init-system-helpers
2. sudo apt install -y qemu-system-x86
3. sudo apt install pass
4. sudo apt install uidmap
5. sudo apt-get install -y gnome-shell-extension-appindicator
6. Now install Docker Desktop from Orientation and setup (https://docs.docker.com/get-started/)

For Ubuntu 18.04, Use Docker Engine (https://docs.docker.com/engine/install/ubuntu/):

1. sudo apt-get install ca-certificates curl gnupg lsb-release
2. sudo mkdir -p /etc/apt/keyrings
3. curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
4. echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
  $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
5. sudo apt-get update
6. sudo apt-get install docker-ce docker-ce-cli containerd.io docker-compose-plugin

Start Docker sample application in the background:
1. sudo docker run -d -p 80:80 docker/getting-started

View docker in webpage:
1. http://localhost:80
